# sama-infra

Terraform modules that define core Sama infrastructure on AWS.

```
ami/ -> AMI for ECS EC2 (deprecated)
bastion/ -> bastion EC2 instance
cdn/ -> CloudFront
db/ -> RDS
ecs/ -> ECS cluster
monitoring/ -> prometheus + grafana
network/ -> VPCs, Subnets, ALBs, etc
```
